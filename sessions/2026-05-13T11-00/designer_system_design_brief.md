# 🎨 데이터시스 소셜 미디어 디자인 시스템 브리프 (v1.0)

## 🎯 목표
모든 Instagram 피드/릴스 콘텐츠에 적용 가능한 통일되고 확장 가능한 비주얼 시스템을 구축하여, 콘텐츠 제작 시간을 단축하고 브랜드 신뢰도를 극대화한다.

## 🖼️ 핵심 요소 (Core Elements)

### 1. 컬러 팔레트 (Color Palette)
| 용도 | 색상 이름 | HEX 코드 | 역할 |
| :--- | :--- | :--- | :--- |
| **Primary** | 데이터시스 메인 | #1A237E | 신뢰, 전문성 (헤더, 주요 배경) |
| **Secondary** | 액센트 블루 | #42A5F5 | 강조, CTA (핵심 지표, 하이라이트) |
| **Neutral** | 라이트 그레이 | #F5F5F5 | 배경, 여백 (가독성 확보) |
| **Dark** | 텍스트/바닥 | #333333 | 기본 텍스트, 다크 모드 요소 |
| **Error** | 경고/오류 | #FF1744 | 오류, 주의사항 (예외 처리 시) |

### 2. 타이포그래피 (Typography)
*   **메인 폰트:** Pretendard (가독성이 높고 디지털 환경에 최적화됨)
*   **활용 원칙:**
    *   **헤드라인 (H1):** Pretendard Bold, 32pt. (최대 정보 전달력)
    *   **서브헤드라인 (H2):** Pretendard SemiBold, 20pt. (주제 구분)
    *   **본문 (Body):** Pretendard Regular, 16pt. (최대 가독성)

### 3. 그리드 및 레이아웃 (Grid & Layout)
*   **기본 그리드:** 10x10 그리드 시스템을 기반으로 모든 레이아웃을 설계한다.
*   **공간 활용:** 최소 여백(Padding)을 20px 이상 확보하여 답답함을 줄이고 전문성을 강조한다.
*   **이미지 배치:** 콘텐츠의 주제를 명확히 할 수 있도록 '정보 차트(Graph)'와 '사람 이미지(Human Element)' 배치를 명확히 구분한다.

## 📐 표준 템플릿 컨셉 3종 (3 Standard Templates)

| 템플릿 명 | 목적 | 적합 콘텐츠 | 시각적 특징 |
| :--- | :--- | :--- | :--- |
| **1. [교육] 개념 정의형 (Concept)** | 복잡한 개념을 단계별로 쉽게 설명 (Carousel) | "AI 자동화란 무엇인가?", "ROI 측정 3단계" | 여백이 많고, 다이어그램/플로우 차트 위주. Secondary 컬러를 활용하여 핵심 개념을 강조. |
| **2. [분석] 사례 제시형 (Case Study)** | 성공 사례, 데이터 기반 분석 결과 제시 (Carousel) | "OO기업의 비용 절감 사례", "데이터 분석 결과 보고서" | 그래프, 차트(Line/Bar Chart)가 지배적. Primary/Dark 컬러를 활용하여 데이터의 신뢰도를 강조. |
| **3. [홍보] 핵심 메시지 강조형 (Impact)** | 서비스의 USP(Unique Selling Point) 또는 핵심 메시지 전달 (Single Post) | "데이터시스가 해결할 수 있는 문제 3가지", "최신 트렌드 요약" | 강렬한 헤드라인(H1)과 큰 비주얼 요소가 중앙에 배치. Primary 컬러를 배경으로 사용, 시선을 즉각적으로 사로잡음. |
