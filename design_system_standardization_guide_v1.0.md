# 🎨 데이터시스 브랜드 디자인 시스템 표준 가이드라인 v1.0

**목표:** 모든 콘텐츠 제작 시 시각적 일관성(Visual Consistency)을 확보하고, 데이터시스의 전문성과 신뢰도를 극대화하는 표준화된 디자인 시스템을 구축합니다.

---

## 1. 🎨 컬러 시스템 (Color Palette)

브랜드의 신뢰성(Trust)과 기술력(Technology)을 상징하는 컬러 팔레트를 정의합니다.

| 역할 | 이름 | Hex Code | RGB (R, G, B) | 용도 및 규칙 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary** | 데이터시스 블루 | `#1A4D99` | (26, 77, 153) | 주 색상. 로고, 핵심 CTA 버튼, 헤드라인 강조. |
| **Secondary** | 테크 그레이 | `#4A5568` | (74, 85, 104) | 보조 텍스트, 구분선, 배경 요소. 전문적인 중립성을 유지. |
| **Background** | 라이트 베이지 | `#F7FAFC` | (247, 250, 252) | 기본 배경색. 순백색 대비 눈의 피로도를 낮춤. |
| **Accent** | 성공/성장 | `#48BB78` | (72, 187, 120) | 긍정적 결과, 성공 지표(ROI), 'OK' 상태 표시. |
| **Error** | 위험/경고 | `#FF1744` | (255, 23, 68) | 실패 지표, 경고 메시지, 'FAIL' 상태 표시. (기존 학습 가이드 유지) |
| **Text** | 기본 텍스트 | `#2D374A` | (45, 55, 74) | 본문 텍스트. 충분한 명암 대비 확보. |

**✅ 컬러 사용 규칙:**
1. Primary 색상은 제목, CTA, 핵심 아이콘에만 제한적으로 사용합니다.
2. Error와 Accent 컬러는 반드시 그 기능적 의미(Failure/Success)를 설명하는 텍스트와 함께 사용해야 합니다.

## 2. ✒️ 타이포그래피 시스템 (Typography System)

기술적 전문성과 가독성(Readability)을 최우선으로 고려하여 계층 구조를 정의합니다.

**🌐 기본 폰트 스택 (Font Stack):**
*   **Web/Digital:** Pretendard, Apple SD Gothic Neo, Malgun Gothic (가장 높은 가독성 확보)
*   **Print/Report:** Noto Sans KR (일관된 전문성 유지)

**📐 폰트 계층 구조 (Hierarchy):**

| 요소 | 크기 (px) | 굵기 (Weight) | 라인 높이 (Line-height) | 사용 목적 |
| :--- | :--- | :--- | :--- | :--- |
| **H1 (페이지 제목)** | 48px | Bold (700) | 1.2 | 핵심 메시지, 백서 제목. 시선 집중. |
| **H2 (섹션 제목)** | 32px | Semi-Bold (600) | 1.3 | 장/섹션 구분. 내용의 큰 덩어리 분리. |
| **H3 (소제목)** | 24px | Semi-Bold (600) | 1.4 | 세부 주제, 리스트 제목. |
| **Body Large** | 18px | Regular (400) | 1.6 | 주요 본문 텍스트. 높은 가독성 확보. |
| **Body Small** | 14px | Regular (400) | 1.5 | 보조 설명, 캡션, 코드 블록 주석. |
| **Caption** | 12px | Regular (400) | 1.4 | 작은 메모, 출처 표기. |

**✅ 타이포 사용 규칙:**
1. 모든 제목은 반드시 Primary 색상이나 Black 계열 텍스트와 조합되어야 합니다.
2. 본문 텍스트는 18px(Body Large)를 기본으로 하여 가독성을 극대화합니다.

## 3. 🖼️ 로고 및 아이콘 사용 규정 (Logo & Iconography)

**A. 로고 사용 규정:**
*   **최소 크기(Minimum Size):** 로고의 폭은 최소 80px 이상을 유지해야 합니다. (훼손 방지)
*   **클리어 스페이스(Clear Space):** 로고 주변에는 로고 높이의 2배 이상을 확보하여 다른 요소와 분리해야 합니다.
*   **금지:** 로고의 색상을 임의로 변경하거나, 늘리거나, 왜곡하는 행위를 엄격히 금지합니다.

**B. 아이콘 시스템:**
*   **스타일:** 모노라인(Monoline) 스타일의 플랫 아이콘을 사용합니다.
*   **크기:** 기본 아이콘 크기는 24x24px로 통일합니다.
*   **컬러:** 아이콘은 기본적으로 Secondary 또는 Primary 색상으로 적용합니다.
